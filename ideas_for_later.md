# Ideas For Later

Park feature ideas here. Never build from this file without an explicit go-ahead.

## Project CRUD in the popup
When we go multi-user, users won't have Supabase dashboard access, so they need a way to add, rename, archive, and delete their own projects from inside the extension. For v1 Greg manages projects directly in Supabase Table Editor.

## Text-only shutdown fallback
Some desktop users won't have a mic. Offer a textbox alongside the record button so they can type the shutdown note instead.
